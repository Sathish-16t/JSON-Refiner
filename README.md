# JSON Refiner

This project converts key-value text into JSON format.

Features
- Convert text to JSON
- Detect numbers and booleans
- Pretty print JSON

Technology Used
- Python
- Gradio

Example Input
name: John Doe
age: 28
is_active: true

Example Output
{
 "name": "John Doe",
 "age": 28,
 "is_active": true
}
